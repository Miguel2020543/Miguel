MIS HABILIDADES

Este blog tiene la finalidad de demostrar algunas de las abilidades que he adquirido durante mis estudios en la universidad. Mas especificamente, en el desarrollo web. Aqui se presentan cada una de las herramientas con las que he trabajado y una breve descripcion de lo que he desarrollado con cada una. Dichas habilidades, fueron puestas en practica al realizar este blog personal.

HTML

Esta es la herramienta con la que mas he trabajado. Cuento con experiencia en el desarrollo de paginas web con esta herramienta y he elaborado desde paginas sencillas, hasta un poco mas complejas como lo es este blog, un menu de inicio, un menu de acceso, entre otros ejemplos mas.

CSS

He trabajado con esta herramienta en conjunto con HTML, ya que se necesitan de ambas para lograr un buen desarrollo de una pagina web. Puedo decir que he empleado esta herramienta para darle color, estilos, tamaño, fuentes y diviones a una pagina web.

JAVASCRIPT

No domino esta herramienta, pero si he desarrollado paginas web con ella, para brindarle movimientos o efectos a ciertos objetos dentro de la pagina web. Dichos objetos pueden ser desde un titulo hasta una imagen.

PHP

Y finalmente php. He empleado esta herramienta para hacer validaciones dentro de una pagina web. Estas validaciones pueden ser empleadas dentro de un formulario, o en un login de inicio de sesion para evitar que cualquier dato pueda ser ingresado.
